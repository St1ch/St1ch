# Cloud

Minecraft plugin developer focused on clean server-side mechanics, quality-of-life systems, and small but polished gameplay tools.

I build plugins for Bukkit, Spigot, Paper, and Folia, mostly around survival mechanics, utility features, and configurable multiplayer experiences.

## Projects

- [Synchro](https://github.com/St1ch/Synchro)  
  Shared-state cooperative survival: synchronized inventories, stats, ender chests, advancements, sleep, and death.

- [Ghost Frames](https://github.com/St1ch/Ghost-Frames)  
  Invisible item frames with container click-through, protection tools, and admin controls.
